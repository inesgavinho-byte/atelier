# @atelier/web

Aplicação web do ATELIER — Next.js 14 (App Router), React 18, TypeScript e Tailwind CSS.

Faz parte do monorepo ATELIER. O modo recomendado de execução é a partir da raiz do repositório, através dos scripts que delegam neste *workspace*:

```bash
# a partir da raiz do repositório
npm install
npm run dev
npm run build
npm run typecheck
npm run lint
```

Em alternativa, é possível executar directamente neste directório:

```bash
# a partir de apps/web
npm run dev
npm run build
```

A especificação do produto encontra-se em `atelier/040-product/AT-0004-product-specification.md`.
