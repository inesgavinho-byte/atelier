# 090 — Objectivos

Objectivos do ATELIER. Cada objectivo é documentado num ficheiro próprio (`GOAL-000X-...`), com critérios de sucesso e artefactos relacionados.

## Como funcionam os objectivos

- Cada objectivo tem um `id` sequencial (`GOAL-0001`, `GOAL-0002`, …).
- O campo `status` indica o estado: Active, Done, Paused ou Dropped.
- O campo `impacts` lista os documentos que o objectivo afecta.
