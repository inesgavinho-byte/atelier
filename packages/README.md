# packages/

Pacotes partilhados e reutilizáveis do monorepo ATELIER (por exemplo, bibliotecas de tipos, componentes de interface ou utilitários comuns a várias aplicações).

Ainda vazio. Os pacotes serão acrescentados quando houver código a partilhar entre aplicações. Os pacotes aqui colocados são automaticamente reconhecidos pelos *workspaces* npm (`packages/*`).
