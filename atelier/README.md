# atelier/ — Fundação Intelectual do ATELIER

Esta pasta é a fonte de verdade da filosofia, organização, decisões e ontologia do ATELIER. Está separada do código da aplicação (`apps/web`), de modo a que o pensamento e o produto evoluam com a mesma durabilidade.

## Gabinetes

| Pasta | Tema |
| ----- | ---- |
| `000-canon/` | Princípios não-negociáveis |
| `010-philosophy/` | Manifesto e intenção fundadora |
| `020-operating-system/` | Sistema operativo intelectual |
| `030-organisation/` | Gabinetes, agentes e responsabilidades |
| `040-product/` | Especificação do produto |
| `050-agent-architecture/` | Arquitectura de agentes |
| `060-decisions/` | Registos de Decisões Arquitecturais (ADR) |
| `070-research/` | Investigação e fontes |
| `080-roadmap/` | Roteiro faseado |
| `090-goals/` | Objectivos |
| `100-ontology/` | Entidades centrais e relações |

## Identificação de documentos

- **AT-000X** — documentos da fundação (Manifesto, Sistema Operativo, Ontologia, …).
- **ADR-000X** — Registos de Decisões Arquitecturais.
- **GOAL-000X** — Objectivos.

Cada documento começa com um bloco de metadados (`id`, `title`, `version`, `status`, `owner`, `created`, `updated`, `depends_on`, `impacts`) seguido das secções **Propósito**, **Âmbito**, **Estado** e **Registo de alterações**.

## Língua

Toda a documentação está em Português Europeu.
